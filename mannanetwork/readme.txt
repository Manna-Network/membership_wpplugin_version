=== The Manna Network Cooperative & Reciprocal Ad Network, Link Directory & Monetization System Wordpress Plugin ===
Contributors: Robert Lefebvre

== Description ==
With this script you can make a complete (with categories, links and management) website directory for your own website and, from there, offer your visitors the opportunity to have their website link info displayed across the entire ad network. They will also have access to this same Plugin (or a PHP script) for their own website. 

== Requirements ==
PHP
CURL
 

== Installation ==
1. Download the plugin and upload it to your website's wp-content/plugins folder.
2. Use the Wordpress admin menu to migrate to the plugins folder and activate the plugin. 
3. Create a page and put the following shortcode where you want the web directory to appear on the page [mannanetwork]
4) Open the page in a browser
